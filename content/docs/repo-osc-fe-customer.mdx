## Key Features

- **Service Discovery**: Browse available services
- **Consultation Booking**: Schedule appointments
- **Complaint Management**: Submit and track complaints
- **Profile Management**: Update personal information
- **Notification Center**: Receive service updates
- **Service History**: View past consultations

## Technology Stack

- Next.js / React
- TypeScript
- Tailwind CSS
- Redux (State Management)
- Axios (API Client)
