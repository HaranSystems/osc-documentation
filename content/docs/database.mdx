# Database

Learn about the database architecture, schema design, and best practices for the OSC system.

## Overview

The OSC system uses a relational database to store:
- User information
- License data
- Application records
- Audit logs
- System configurations

## Schema Design

### Core Tables

- **Users**: User accounts and authentication
- **Licenses**: License management and tracking
- **Applications**: Registered applications
- **Audit Logs**: System activity tracking
