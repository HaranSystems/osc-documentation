## Repository Categories

### Backend Services (6 repositories)
- **osc-be0-utiliti**: Shared utilities and common functions
- **osc-be1-admin-profile**: User profile and admin management
- **osc-be2-pelesenan**: Consultation and scheduling services
- **osc-be3-aduan-dan-notifikasi**: Complaint handling and notifications
- **osc-be5-enforcement**: Enforcement and compliance management

### Frontend Applications (2 repositories)
- **osc-fe-admin**: Administrator web interface
- **osc-fe-customer**: Customer web interface

### Mobile & Infrastructure (2 repositories)
- **osc-mobile**: Cross-platform mobile application
- **k8s-config**: Kubernetes configuration and infrastructure
